# Some user

This project was made for the purpose of implementing an api similar to <https://randomuser.me/>.

## Prerequisites

- .Net Core 3.1

## Getting started

Clone the repository.

```bash
git clone https://...
```

Change directory to the API project folder.

```bash
cd src/SomeUser.Api
```

Start the web api on <http://localhost:8080>.

```bash
dotnet run
```
